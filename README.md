# Donatech — Hub Logístico de Donaciones

Plataforma de microservicios para centralizar la logística de donaciones ante catástrofes naturales en Chile.

## Estructura del monorepo

| Carpeta | Estado | Descripción |
|---|---|---|
| `Backend/` | Completo | 9 microservicios Spring Boot + Docker Compose |
| `Frontend/` | En desarrollo | React + Vite + pnpm |
